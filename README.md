## 🧠 What is Caption Crafter?

**Caption Crafter** is a user-friendly web application powered by artificial intelligence that generates descriptive captions for uploaded images. Whether you're a content creator, digital marketer, student, or simply curious – this tool helps you turn visuals into text instantly.

## ✨ Key Features

- 📤 **Image Upload** – Drag and drop or select any image from your device.
- 🤖 **AI-Powered Captioning** – Get a smart, context-aware caption generated using machine learning models.
- ⚡ **Instant Results** – Lightning-fast processing and response time.
- 🎨 **Clean UI** – Simple, aesthetic interface for all users.
- 📱 **Mobile Responsive** – Optimized for both desktop and mobile use.

## 💻 Tech Stack

- **Frontend**: HTML, CSS, JavaScript (likely using React)
- **Backend**: AI-based image captioning model (Python with Flask)

## 📦 How to Use

1. Go to the live app: [Caption Crafter](https://polite-mandazi-d51ffd.netlify.app)
